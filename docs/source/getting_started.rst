Getting Started
===============

Installation
------------

Install from PyPI:

.. code-block:: bash

   pip install SILEXlight

Install in editable mode for development:

.. code-block:: bash

   pip install -e .[test]

Run tests
---------

.. code-block:: bash

   python -m pytest --pyargs SILEXlight.tests

Examples
--------

For complete examples covering gmsh IO, TRI3, TET4, and Fortran backends,
see ``Complete Examples (All Libraries)`` in the documentation.

API Notes
---------

The API reference pages are generated from function docstrings in the core
Python modules and from routine-purpose comments in the Fortran sources.
